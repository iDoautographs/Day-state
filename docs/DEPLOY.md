GitHub Pages deploy:

1. Create a new GitHub repo named day-state.
2. Push all files from this folder to the repo.
3. In GitHub: Settings -> Pages.
4. Set Source to GitHub Actions.
5. Push to main.
6. Your site will publish.

Live URL:
https://idoautographs.github.io/day-state/

Notes:
- This project is fully static.
- Browser data is saved with localStorage on that device.
- For cross-device sync later, add a backend.
